Project Description
An Android application demonstrating multi-activity layout design, implicit intents, custom options menu synchronization via a base inheritance architecture (`SamuelBaseActivity`), dynamic image rendering tracks, and background lifecycle verification utilizing explicit Logcat tracking.

Project's Purpose
-Screen 1 (`SamuelActivity11`):Portrait and Landscape layout configurations, dynamic input validation, and persistent indefinite Snackbar controls.
-Screen 2 (`PecoraActivity22`):Automatic bundle long integer extraction and a synchronized index pointer looping system icons.
-Shared Options Menu:Implicit phone dialer launching, school landing page external redirect execution, and clean application closure routines without code repetition.
